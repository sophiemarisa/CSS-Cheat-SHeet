CSS Snippet Cheatsheet

Use these CSS snippets in your own code!

Flexbox Row

Use these three properties to make a Flexbox row layout.

.row{
  display: flex;
  flex-direction: row;
  flex-wrap: wrap;
}

Flexbox Column

Use this to make a Flexbox column layout.

.column{
  display: flex;
  flex-direction: column;
}

CSS Grid Layout

Build a 12 column layout using a CSS grid.

.grid{
  display: grid;
  width: 100%;
  grid-template-columns:
repeat(12, 1fr);
}
